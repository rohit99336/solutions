Ubuntu में `apt` के ज़रिए इंस्टॉल किए गए **Flameshot** को अनइंस्टॉल करने के लिए आप नीचे दिए गए स्टेप्स फॉलो करें:

### 1. ऐप को अनइंस्टॉल करने का कमांड

Terminal खोलें और यह कमांड टाइप करें:

```bash
sudo apt remove flameshot

```

*(यह कमांड ऐप को हटा देगा, लेकिन उसकी कॉन्फ़िगरेशन फाइल्स सिस्टम में छोड़ देगा।)*

### 2. ऐप और उसकी सेटिंग्स पूरी तरह हटाने के लिए (Recommended)

अगर आप चाहते हैं कि Flameshot की सेटिंग्स और कॉन्फ़िगरेशन फाइल्स भी पूरी तरह साफ़ हो जाएं, तो **purge** का इस्तेमाल करें:

```bash
sudo apt purge flameshot

```

### 3. बची-कुची फाइल्स (Dependencies) साफ़ करने के लिए

ऐप हटाने के बाद, कुछ ऐसी फाइल्स रह जाती हैं जिनकी अब ज़रूरत नहीं है। उन्हें हटाने के लिए यह कमांड चलाएँ:

```bash
sudo apt autoremove

```

---

### संक्षेप में (Quick Steps):

बस इन दो कमांड्स को एक के बाद एक चलाएँ:

1. `sudo apt purge flameshot -y`
2. `sudo apt autoremove -y`

**नोट:** `sudo` कमांड चलाने के बाद आपसे पासवर्ड माँगा जाएगा। टाइप करते समय पासवर्ड स्क्रीन पर दिखेगा नहीं (यह Linux का सिक्योरिटी फीचर है), बस पासवर्ड टाइप करके **Enter** दबा दें।

क्या आप स्क्रीनशॉट के लिए किसी दूसरे टूल (जैसे Spectacle या Built-in Screenshot) का इस्तेमाल करना चाहते हैं?

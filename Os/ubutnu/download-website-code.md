Ubuntu (Linux) तो डेवलपर्स के लिए सबसे बेहतरीन OS है और इसमें वेबसाइट क्लोन/डाउनलोड करने के लिए बहुत पावरफुल टूल्स पहले से मौजूद हैं।

### 1. HTTrack (सबसे लोकप्रिय और बेस्ट)

यह वही टूल है जिसका शायद आप "wttracker" नाम से जिक्र कर रहे थे। यह पूरी वेबसाइट (HTML, CSS, Images, JS) को आपके कंप्यूटर पर ऑफलाइन सेव कर देता है।

**Ubuntu में Install कैसे करें:**
Terminal खोलें (`Ctrl+Alt+T`) और यह कमांड चलाएं:

```bash
sudo apt update
sudo apt install httrack webhttrack

```

**कैसे यूज़ करें:**

1. इंस्टॉलेशन के बाद टर्मिनल में `webhttrack` टाइप करें।
2. यह आपके ब्राउज़र में एक GUI इंटरफ़ेस खोल देगा।
3. वहां प्रोजेक्ट का नाम और वेबसाइट का URL डालें।
4. यह पूरी साइट डाउनलोड करके आपके folder में सेव कर देगा।

---

### 2. Wget (सिर्फ एक कमांड से काम तमाम) ⚡

Ubuntu में `wget` पहले से होता है। यह सबसे fast और powerful तरीका है। आपको कोई एप इनस्टॉल करने की भी जरूरत नहीं है।

**Terminal कमांड:**
जिस वेबसाइट को डाउनलोड करना है, उसके लिए यह कमांड चलाएं:

```bash
wget --mirror --convert-links --adjust-extension --page-requisites --no-parent https://example.com

```

**इस कमांड का मतलब:**

* `--mirror`: पूरी वेबसाइट की कॉपी बनाएगा।
* `--convert-links`: लिंक्स को ऐसे बदल देगा ताकि वे ऑफलाइन (बिना इंटरनेट) भी चलें।
* `--page-requisites`: CSS, JS और Images भी डाउनलोड करेगा।
* `--no-parent`: यह सुनिश्चित करेगा कि वह main folder के बाहर की फाइलें डाउनलोड न करे।

### 3. Online Website (बिना कुछ इनस्टॉल किए)

अगर आप कोई सॉफ्टवेयर इनस्टॉल नहीं करना चाहते, तो आप इस वेबसाइट का यूज़ कर सकते हैं:

* **Website:** `saveweb2zip.com`
* **कैसे यूज़ करें:** बस वेबसाइट का URL डालें, यह पूरी साइट को स्कैन करके आपको एक **ZIP file** दे देगा जिसे आप डाउनलोड कर सकते हैं।

### ⚠️ ज़रूरी नोट (Technical Limitations)

चूंकि आप एक डेवलपर हैं, आपको यह जानना ज़रूरी है:
ये सभी टूल्स सिर्फ़ **Frontend (HTML, CSS, JavaScript, Images)** डाउनलोड कर सकते हैं।

ये टूल्स **Backend Logic (PHP Code, Laravel Controllers, Database, .env file)** को डाउनलोड **नहीं** कर सकते। सर्वर-साइड कोड (PHP) कभी भी ब्राउज़र को नहीं दिखता, इसलिए उसे कॉपी नहीं किया जा सकता।

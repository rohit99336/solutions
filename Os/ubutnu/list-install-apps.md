Ubuntu (Linux) में इंस्टॉल किए गए ऐप्स की लिस्ट देखने के कई तरीके हैं, जो इस बात पर निर्भर करते हैं कि आपने उन्हें कैसे इंस्टॉल किया था (Terminal से या Software Center से)।

यहाँ सबसे आसान और प्रभावी कमांड्स दी गई हैं:

### 1. सभी इंस्टॉल किए गए पैकेजेस देखने के लिए (APT)

अगर आपने `apt install` का उपयोग किया है, तो यह कमांड सबसे बेस्ट है:

```bash
apt list --installed

```

*चूंकि यह लिस्ट बहुत लंबी हो सकती है, इसलिए आप इसे **'less'** के साथ इस्तेमाल करें ताकि आप आराम से स्क्रॉल कर सकें:*

```bash
apt list --installed | less

```

### 2. Snap Apps देखने के लिए

आजकल Ubuntu में बहुत सारे ऐप्स (जैसे VS Code, Spotify) **Snap** के ज़रिए इंस्टॉल होते हैं। उन्हें देखने के लिए टाइप करें:

```bash
snap list

```

### 3. सिर्फ वो ऐप्स जो आपने खुद इंस्टॉल किए (GUI Apps)

अगर आप सिस्टम फाइल्स और लाइब्रेरीज़ को छोड़कर सिर्फ उन ऐप्स को देखना चाहते हैं जिन्हें आप "Apps Menu" में देखते हैं, तो यह कमांड चलाएँ:

```bash
ls /usr/share/applications

```

### 4. Flatpak Apps (अगर आप यूज़ करते हैं)

अगर आपने Flatpak सेटअप किया हुआ है, तो इसकी लिस्ट यहाँ से मिलेगी:

```bash
flatpak list

```

---

### 5. ग्राफिकल तरीका (बिना कमांड के)

अगर आप टर्मिनल यूज़ नहीं करना चाहते:

1. **Ubuntu Software Center** खोलें।
2. ऊपर दिए गए **'Installed'** टैब पर क्लिक करें।
3. यहाँ आपको उन सभी मुख्य ऐप्स की लिस्ट मिल जाएगी जिन्हें आप अनइंस्टॉल भी कर सकते हैं।

### प्रो टिप (Search करना):

अगर आप किसी खास ऐप को ढूंढ रहे हैं (जैसे 'chrome'), तो आप **grep** का इस्तेमाल कर सकते हैं:

```bash
dpkg -l | grep -i chrome

```

**क्या आप इनमें से किसी ऐप को अनइंस्टॉल करने का तरीका भी जानना चाहते हैं?**

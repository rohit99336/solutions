आपका लैपटॉप Ubuntu पर है और मोबाइल के हॉटस्पॉट से जुड़ा है, तो अपने Laravel 12 प्रोजेक्ट को मोबाइल में चलाने के लिए ये स्टेप्स फॉलो करें:

### 1. लैपटॉप का IP एड्रेस पता करें

सबसे पहले अपने Ubuntu लैपटॉप का लोकल IP एड्रेस निकालें:

* **Terminal** (`Ctrl+Alt+T`) खोलें।
* यह कमांड टाइप करें: `hostname -I`
* आपको एक नंबर दिखेगा जैसे: **192.168.43.10**। इसे नोट कर लें।

### 2. Laravel सर्वर को नेटवर्क के लिए चालू करें

अगर आप सिर्फ `php artisan serve` लिखेंगे, तो वह मोबाइल पर नहीं खुलेगा। आपको इसे ऐसे चलाना होगा:

* अपने प्रोजेक्ट फोल्डर में जाकर टर्मिनल में यह कमांड लिखें:
`php artisan serve --host=0.0.0.0 --port=8000`
* **0.0.0.0** का मतलब है कि अब आपका लैपटॉप नेटवर्क के दूसरे डिवाइसेस (जैसे मोबाइल) से रिक्वेस्ट ले सकता है।

### 3. Ubuntu Firewall को परमिशन दें

Ubuntu में सुरक्षा के लिए अक्सर पोर्ट्स बंद होते हैं। पोर्ट 8000 को खोलने के लिए यह कमांड चलाएं:
`sudo ufw allow 8000/tcp`

### 4. मोबाइल में प्रोजेक्ट खोलें

* अब अपने मोबाइल का **Chrome** ब्राउज़र खोलें।
* एड्रेस बार में अपने लैपटॉप का IP और पोर्ट नंबर डालें। उदाहरण के लिए:
`http://192.168.43.10:8000`
*(यहाँ `192.168.43.10` की जगह अपना वाला IP डालें)*

### **Laravel 12 (Vite) के लिए जरूरी टिप:**

अगर मोबाइल पर वेबसाइट खुल रही है लेकिन CSS या डिज़ाइन खराब दिख रहा है, तो आपको **Vite** को भी कॉन्फ़िगर करना होगा:

1. अपनी `vite.config.js` फाइल खोलें।
2. उसमें `server` वाला हिस्सा ऐसे जोड़ें:

```javascript
export default defineConfig({
    server: {
        host: '0.0.0.0',
    },
    // बाकी कोड...
});

```

3. इसके बाद टर्मिनल में `npm run dev` चलाएं।

अब आपके मोबाइल पर CSS और JS भी सही से काम करेंगे।

क्या आपका मोबाइल कनेक्ट हो गया? अगर कोई और एरर आ रहा है तो बेझिझक पूछें।

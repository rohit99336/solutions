# 🌿 Grafting कितने प्रकार के होते हैं?

👉 कुल मिलाकर **10–15 main grafting methods** होते हैं
लेकिन practically **6–8 methods सबसे ज्यादा use होते हैं**

---

# 🌳 सबसे common grafting methods

## 1. ✂️ **Cleft Grafting**

* मोटी branch को बीच से split करके scion लगाते हैं
  ✔️ बड़े पेड़ में सबसे useful

---

## 2. 🔺 **Wedge Grafting**

* scion को V (wedge) shape में काटकर लगाते हैं
  ✔️ hibiscus, mango आदि में common

---

## 3. 🌿 **Approach Grafting**

* दोनों plants जुड़े रहते हैं
  ✔️ सबसे safe method

---

## 4. 🌱 **Side Grafting**

* side में cut करके scion insert करते हैं
  ✔️ ornamental plants में use

---

## 5. 🌳 **Bark Grafting**

* छाल (bark) को उठाकर scion लगाते हैं
  ✔️ बहुत मोटे तने (old trees) में best

---

## 6. 🔄 **Whip & Tongue Grafting**

* दोनों parts same thickness के होते हैं
  ✔️ nursery plants में use

---

## 7. 🌼 **Bud Grafting (Budding)**

* पूरा branch नहीं, सिर्फ **एक bud (कली)** लगाई जाती है
  ✔️ rose, citrus में बहुत common

---

## 8. 🌿 **Crown Grafting**

* trunk के ऊपर कई graft एक साथ
  ✔️ tree top बदलने के लिए

---

# 📊 आसान भाषा में समझें

👉 3 main categories होती हैं:

### 1. 🌿 Branch Grafting

* cleft, wedge, bark, side

### 2. 🌼 Bud Grafting

* T-budding, chip budding

### 3. 🌳 Special Grafting

* approach, crown

---

# 🔥 कौन सा method कब use करें?

| Situation          | Best Method  |
| ------------------ | ------------ |
| मोटी branch        | Cleft / Bark |
| पतली branch        | Wedge / Whip |
| beginner           | Approach     |
| multi variety tree | Cleft + Side |
| rare plant         | Approach     |

---

# 👍 Final समझ

👉 theoretically 10–15 methods हैं
👉 practically आपको सिर्फ ये याद रखना है:

✔️ **4–5 method सीख लो = हर plant graft कर सकते हो**
